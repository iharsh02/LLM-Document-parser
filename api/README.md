# Backend API

This is the backend API for the Hela project, built with TypeScript and Bun.

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) installed on your machine.

### Installation

1. Install the dependencies:

    ```bash
        bun install
    ```

### Running the Application

- **Development Mode:**

    ```bash
    bun dev
    ```

- **Production Mode:**

    ```bash
     bun start
    ```
